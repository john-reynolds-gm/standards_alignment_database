# EM2 Standards Alignment Explorer

Streamlit app for exploring Learnosity alignment data between Eureka Math 2 state standards and CCSS.

## Prerequisites

- Python 3.10+
- `learnosity.db` in the same directory as `app.py` — generated by `build_database.py`

## Running locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

The app opens automatically at `http://localhost:8501`.

## Deploying to Streamlit Community Cloud

Streamlit Community Cloud hosts apps for free from a public or private GitHub repo.

1. Make sure `learnosity.db` is committed to the repo alongside `app.py` and `requirements.txt`.
2. Go to [share.streamlit.io](https://share.streamlit.io) and sign in with GitHub.
3. Click **Create app** → **Deploy a public app from GitHub**.
4. Select your repo, branch, and set **Main file path** to `app.py`.
5. Click **Deploy**. The app will be live at a `*.streamlit.app` URL within a minute or two.
6. Share that URL with colleagues — no login required to view.

> **Updating the database:** After running `build_database.py` locally, commit and push the new `learnosity.db`. Streamlit Community Cloud detects the push and redeploys automatically.

## Adding UC3 later

When the similarity predictions and alignment guide data are ready, UC3 can be added as a third tab in `app.py` without touching the existing tab code. The three planned database tables (`similarity_candidates`, `state_alignment_guide`, `ccss_alignment_guide`) are already in the schema design — load them by extending `build_database.py`.
